# Проект "Интернет-магазин"

## Описание:
1. Инициализирован новый проект Django внутри проекта online_shop.
2. Создано новое приложение с названием catalog.
3. Созданы два HTML-шаблона: для домашней страницы и для страницы с контактной информацией,
для стилизации страниц использовался Bootstrap.
4. Создан контроллер для отображения домашней страницы.
5. Создан контроллер для отображения страницы с контактной информацией.
6. Настроена маршрутизация для контроллеров.
7. Создан новый контроллер и шаблон для отображения страницы с подробной
информацией о товаре. На этой странице показана вся информация о товаре.
8. Добавлен в шаблон главной страницы код для отображения списка товаров
с помощью цикла. Чтобы карточки товаров выглядели одинаково, обрезано
отображаемое описание до первых 100 символов.
9. Выделен общий (базовый) шаблон, который включает общие элементы страницы,
такие как шапка, подвал и стили. Также создан подшаблон для главного меню,
который включается в другие шаблоны.
10. Создано новое приложение для блога.
11. Переведены имеющиеся в проекте контроллеры с FBV на CBV.
12. Добавлена логика на уровне контроллеров:
 - Увеличение счетчика просмотров
 - Фильтрация опубликованных статей
 - Перенаправление после редактирования
13. Реализован механизм CRUD для модели продуктов, используя модуль django.forms.
При этом создано ограничение на присваивание продуктам названия и добавления в описание слова, которые
включены в список запрещенных слов.
14. Реализована валидация формы, чтобы проверять отсутствие этих слов (в любом регистре) полях.
15. Добавлена валидация для поля price (цена не может быть отрицательной).
16. Добавлена стилизация форм для продуктов.
17. Создано новое приложение для работы с пользователями.
18. В сервисе реализована функция аутентификации.
19. Закрыт доступ к страницам, связанным с управлением продуктами, для анонимных пользователей.
20. Создана группа для роли «Модератор продуктов» и настроены необходимые разрешения.
21. Реализовано решение, которое проверяет, что редактирование и удаление продукта доступно только
его владельцу.


## Установка:

1. Клонируйте репозиторий:
```
git@github.com:Elena-Kandrushina/online_shop.git
```

2. Установите зависимости:
```
poetry install
```
