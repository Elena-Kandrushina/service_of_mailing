# Проект "Сервис управления рассылками"

## Описание:
Создано веб-приложение на Django, которое позволяет пользователям управлять рассылками сообщений
для клиентов. Приложение включает функциональность для создания, просмотра, редактирования и
удаления рассылок, а также отправки сообщений по требованию.
Выполнено:
1. Управление клиентами
2. Управление сообщениями
3. Управление рассылками
4. Отправка сообщений по требованию
5. Попытки рассылок и сохранение статистики
6. На главной странице отображаются данные о количистве рассылок, уникальных получателей и количество
активных рассылок
7. Регистрация и аутентификация пользователей
8. Статистика и отчеты
9. Ограничение доступа
10. Кеширование

## Установка:

1. Клонируйте репозиторий:
```
git@github.com:Elena-Kandrushina/service_of_mailing.git
```

2. Установите зависимости:
```
poetry install
```
3. Запуск Redis:
```
redis-server
redis-cli

```

4. Запуск проекта после запуска Redis:
```
python manage.py runserver
```
